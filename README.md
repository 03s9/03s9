## Featured Project

### [Java HTTP Server + React Frontend](https://github.com/[seu-usuario]/java-http-server)

> Production-grade REST API built from scratch in Java 17 — no Spring, no frameworks.

**What it does:**
- HTTP server with TCP socket handling and `ExecutorService` thread pool
- Full CRUD REST API with path params, query params, and paginated responses
- JWT authentication (1h access token + 7-day refresh token stored as SHA-256 in PostgreSQL)
- BCrypt (12 rounds) password hashing · Role-based access control (admin/user)
- In-memory rate limiting — 5 login attempts/min with sliding window (`ConcurrentHashMap`)
- React 18 frontend with Axios interceptors for automatic token refresh, pagination, and inline editing
